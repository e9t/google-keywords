Get Google Keywords
=================================

### Preparation

1. 다음 패키지들을 설치한다.
- Mac OS X:

        sudo pip install nltk
    
- Windows: Install [NLTK](http://nltk.org/install.html) 

### Setup

1. 설정 파일을 변경한다.

        vi settings.py

### Run

1. 스크립트를 실행한다.
- 영문
    
        python crawlers/crawler.py 'query here'
        python utils/keywords.py 'query here'
